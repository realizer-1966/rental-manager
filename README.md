# 임대관리 시스템 (Rental Manager)

Flask + SQLite 기반 임대관리 웹 애플리케이션입니다.
Termux/Android 환경에서 실행됩니다.

## 주요 기능

### 건물/호수 관리
- 건물 등록 (원룸건물, 투룸건물, 오피스텔, 분양형호텔, 복합건물)
- 호수 관리 (층: 숫자/B1/B2/반지하 텍스트 입력, 평수, 보증금, 월세, 관리비)
- 공실/임대중/수리중 상태 관리

### 임차인 관리
- 임차인 CRUD (이름, 전화, 이메일, 주민/사업자번호, 비상연락처)
- 이름/전화번호 검색
- 활성 계약 수 표시

### 계약 관리
- 월세/전세/반전세 계약 등록
- 퇴실 청소비, 1인 추가 비용 기록
- 납부일 변경 (변경 이력 자동 기록, 최초 납부일 보존)
- 계약 수정 (보증금, 월세, 관리비, 청소비, 종료일, 납부일)

### 보증금 분할 납부
- 보증금을 나누어 납부하는 경우 추적
- 납부 이력 관리, 잔액 표시, 완납 확인

### 월세 청구/수납
- 월별 청구 일괄 생성
- 수납 등록 (입금자명, 입금메모/호수, 수납방법)
- 부분납/완납 자동 판정
- 연체 자동 감지

### 퇴거 정산
- 일할 월세 계산 (월세/30일 기준)
- 전기/가스 요금 정산
- 퇴실 청소비 정산
- 추가 공제 (수리비, 위약금 등)
- 보증금 분할 납금 반영 (실제 납부액 기준 반환)
- 정산 미리보기 + 확정

### 계량기 관리
- 입주/퇴거 시 전기/가스 계량기 기록
- 계량기 사진 업로드 (카메라 직접 촬영 가능)
- 수치 기록 (kWh, m³)

### 공과금 관리
- 한전/도시가스 청구 기록
- 문자(SMS/알림톡) 원본 보관
- 수납 등록, 미납 추적

### 수리 내역
- 호수별 수리 기록 (전등, 에어컨, 보일러, 도어록, 배관, 도배 등)
- 건물별 수리 기록 (옥상, 외벽, 엘리베이터, 공동구 등)
- 수리 비용, 수리업체, 상태(접수/진행중/완료)
- 수리 사진 업로드

### 계약서 OCR
- 계약서 사진 업로드 → Tesseract OCR로 텍스트 추출
- 자동 항목 파싱 (보증금, 월세, 관리비, 청소비, 1인추가비용, 계약기간, 임차인명, 납부일)
- 추출 결과를 계약에 일괄 반영
- OCR 처리 이력 보관

### 장부 (은행 입출금)
- 은행 SMS/알림톡 붙여넣기 → 자동 파싱 (국민, 신한, 카카오, 하나, 우리, 농협)
- 입금자명, 금액, 메모 자동 추출
- 수동 입력 지원
- 미매칭 입금 → 월세 수납 자동 매칭
- 입출금 통계 (총 입금, 총 출금, 잔액)

### 대시보드
- 총 건물/호수, 임대중/공실, 가동률
- 이번달 수납액, 연체 건수/금액
- 건물별 현황, 연체 현황

## 설치 및 실행

```bash
# Termux 환경
pip install flask werkzeug pytesseract Pillow requests

# Tesseract OCR + 한국어 팩
pkg install tesseract
# tessdata 디렉토리에 kor.traineddata 다운로드

# 실행
python app.py
# http://localhost:5050 접속
# 기본 관리자: admin / admin123
```

## 기술 스택
- Python 3 + Flask
- SQLite
- Tesseract OCR (한국어+영어)
- Bootstrap 5
- Termux/Android
